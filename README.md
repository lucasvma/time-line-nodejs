# **TimeLine Project** #
* * *
> A simple Timeline with NodeJS, React and MongoDB

&nbsp;

## **Development** ##
* * *
* ### **Prerequisites** ###
    - Install [Node.js](https://nodejs.org)
    - Install [npm](https://www.npmjs.com)

* ### **Clone the repo** ###
    `$ git clone https://github.com/lucasvma/time-line-nodejs.git`

* ### **Install dependecies** ###
    `$ npm install`

* ### **Run** ###
    `$ npm start`
